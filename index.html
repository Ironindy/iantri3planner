⚠️ 데이터 저장 안내

이 앱은 브라우저 내부 저장 방식을 사용합니다.
입력한 데이터는 현재 사용 중인 브라우저와 기기에서만 저장됩니다.

예를 들어 아래와 같이 환경이 바뀌면 기존 데이터가 보이지 않습니다.

· 크롬 → 삼성 브라우저
· PC → 스마트폰
· 카카오톡/밴드 내 브라우저

권장 사용 방법

1. 홈 화면에 바로가기를 만들어 실행하세요.
2. 항상 같은 브라우저에서 사용하세요.
3. 데이터 입력 후 파일 저장(백업)을 해두세요.
4. 다른 환경에서 사용할 때는 파일 불러오기로 데이터를 가져오세요.
